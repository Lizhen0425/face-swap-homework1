# face-swap-homework1
2025年工程數學作業實作。本作業實作人臉置換（Face Swap）流程，利用 InsightFace 套件的臉部偵測與特徵比對功能，搭配 InSwapper 模型，將來源人臉自動套用到影片中出現的人物上，並合併音訊輸出完整影片。目的在於熟悉電腦視覺與模型推論流程。


# 遇到問題
- 極端狀況
影片2(AI-ubereat)裡的人臉角度太斜（大約轉超過60度）時，模型常常抓不到臉，或是有抓到但辨識結果很不穩。這時候程式就不會換臉，看起來會像那幾幀突然沒換到。這應該是臉的角度太偏導致特徵抓不準。

- 遮擋（像手擋住臉或戴口罩）
當臉被手遮住時，相似度會變得很低，低到設定的門檻（0.30）以下，程式就自動跳過不換。影片裡有時候會看到人臉一換一不換，主要是因為遮擋讓模型認不出來，影片1、影片3就有遇到，因為他的手一直揮。

- 快速移動或模糊
影片中如果人物動作太快、畫面模糊，模型抓臉的框有時候會抖一下，導致臉貼回去的邊緣有一點飄移。這個情況在換臉影片1(AI-wang)、2(AI-ubereat)轉頭那幾幀很明顯。
